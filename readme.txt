Easily remove any/all of the default WordPress dashboard widgets

== Description ==

Adds an options page for an Administrator to choose which default WordPress dashboard widgets are available.

== Installation ==

1. Upload `clean-wp-dashboard.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure dashboard via the new "Dashboard Settings" admin page
